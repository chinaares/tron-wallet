## grpc and apk build
```
./gradlew build
```

## Features

- Register wallet
- Import wallet
- Send Coin
- Receive Coin
- My Tron balance
- My Frozen balance
- Freeze
- Unfreeze
- Vote representative
- Token list
- Representative list
