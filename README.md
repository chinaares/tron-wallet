## Required.
 - Java JRE 1.8
 
## grpc and apk build
```
./gradlew build
```

## Run
Download link: https://github.com/leckylao/tron-wallet/blob/win_wallet/TronWinWallet.zip

Open Run.bat

## Features

- Register wallet
- Import wallet
- Send Coin
- Receive Coin
- My Tron balance
- My Frozen balance
- Freeze
- Unfreeze
- Vote representative
- Token list
- Representative list
